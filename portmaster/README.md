# godot3导出游戏到吹米(TrimUI Smart Pro)掌机
环境:吹米(TrimUI Smart Pro)掌机，固件版本1.0.4，系统ui是CrossMix-OS，portmaster里面安装了godot3运行环境  
使用方式:将battle-city.sh脚本和battle-city文件复制到SDCARD:/Data/ports目录下面，然后在引擎中导出坦克大战  
的linux程序，只需要pck，架构选择arm64，纹理勾选etc2，资源里面导出*.json，然后将pck复制到battle-city\gamedata  
文件夹里面改名为battle-city.pck，之后就可以在掌机文件管理器里面运行脚本battle-city.sh
