# battlecity-clone
godot battlecity game clone  
itch.io https://absolve.itch.io/battle-city-remake  
  
Controls
Player 1:  arrows and J to fire  
Player 2:  WSAD and keypad 3 to fire  
Select: ENTER  

导出是需要将地图文件一起导出，就是json的那些文件
